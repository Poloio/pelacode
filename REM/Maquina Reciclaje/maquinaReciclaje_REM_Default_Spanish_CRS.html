UC-0001 Reciclar elementos
Versión 1.0 ( 25/11/2019 )
Autores ?
Fuentes ?
Dependencias Ninguno
Descripción El sistema deberá comportarse tal como se describe en el siguiente caso de uso cuando un Usuario quiere reciclar unos elementos.
Precondición PD
Secuencia normal Paso Acción
1 El sistema inicia el proceso y muestra información al Usuario con los objetos que va introduciendo. Se muestra también una opción de cancelar operación.
2 El actor Usuario (ACT-0001) introduce un elemento a reciclar.
3 Si el objeto es uno de los definidos como reciclable, el sistema añade el objeto a la lista. Vuelve a paso 1.
4 El actor Usuario (ACT-0001) termina de introducir objetos y selecciona "terminar" para recibir su recibo con el dinero.
5 El sistema calcula el dinero a pagar, imprime el recibo y entrega el dinero.
Postcondición PD
Excepciones Paso Acción
3 Si no es un objeto definido, el sistema devuelve dicho objeto al Usuario y vuelve al paso 2., a continuación este caso de uso PD
